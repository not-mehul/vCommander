--- Changelog ---
v3.0
- Revamped GUI, new frontend built with Flet
- Restructured backend to support long-term iteration
- Added Commission Tool
- Version check on startup with Github

v3.1
- (New feature) Sort the copy button for the invited users in alphabetical order
- (Bug fix) Updated the subdomain for vproresponse to vagent
- (Bug fix) Added LPR camera door linking to the commission tool

v3.2
- Alarms now creates a response, setting it to Self-monitored mode.
- AI powered search and certain additional analytics also toggled on by default
- Alarms Specialist Commission will now allow toggling face analytics, with the default toggled to True.
- (Improvement) Decommission tool overhaul: deletes the full asset set in an
  explicit dependency order (devices, then Access Control, then Alarms),
  with Access Control and Alarms grouped under collapsible parent checkboxes
- (Improvement) Decommission now shows serial numbers alongside object ids
  in the asset list, CSV export, and progress rows
- (Improvement) Each deletion writes a detailed system message to api_calls.log
- (Improvement) Decommission now deletes Access Station Pro devices (after
  doors, before access controllers)
