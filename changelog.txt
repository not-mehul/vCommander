--- Changelog ---
v3.0
- Revamped GUI, new frontend built with Flet
- Restructured backend to support long-term iteration
- Added Commission Tool
- Version check on startup with Github

v3.1
- (New feature) Sort the copy button for the invited users in alphabetical order
- (Bug fix) Updated the subdomain for vproresponse to vagent
- (Bug fix) Added LPR camera door linking to the commission tool

v3.2
- (New feature) Decommission tool deletes granular assets and entities: Access Groups, Access Levels, Doors, Floors, Buildings, Visitor access, Alarm Systems, Alarm Partitions, Alarm Guards, Alarm Wired Inputs/Outputs
- (New feature) Decommission tool CSV export shows the serial numbers alongside object ids wherever present.
- (New feature) Decommission tool now lists and deletes Access Station Pro devices
- (Bug fix) Alarms now creates a response, setting it to Self-monitored mode.
- (Bug fix) Decommission tool now deletes Alarm Input/Output devices.
- (Bug fix) AI powered search and certain additional analytics also toggled on by default
- (Bug fix) Alarms Specialist Commission will now allow toggling face analytics, with the default toggled to True.
- (Bug fix) Alarms Specialist creates a General Keycode for the Instructor - 123456
- (Bug fix) Session timeout is now a fixed 30-minute window from login.

v3.3
- (UI) Decommission progress collapses to one row per category with a live
  "n / m deleted" counter; per-item detail is one click away
- (UI) Decommission and Commission both support Cancel: the in-flight item
  completes, the rest of the run is skipped, and the summary shows where
  it stopped
- (UI) Decommission SELECT gains a Select-all / Select-none top bar, live
  "(N selected / M total)" chips on every group and category header, a
  search filter (name or serial), and a Show-items toggle that defaults to
  the compact one-line-per-category layout
- (UI) Non-fatal alerts (export complete, validation hints, copy success)
  are now non-blocking bottom-center toasts instead of modal dialogs
- (UI) Login form now uses two columns (email|password, org|region+shard)
  and submits on Enter from any field
- (UI) Global keyboard shortcuts: Esc backs out, Cmd/Ctrl-K jumps to Home,
  Cmd/Ctrl-, logs out
- (UI) TEXT_SECONDARY contrast bumped from #a0a0a0 to #b8b8b8 (~5:1
  contrast over the dark background, passes WCAG AA for small body text)


--- NEWS ---
- When the Alarm is triggered, you will not be able to delete an Alarm Site. It needs to be resolved to delete an alarm site.
