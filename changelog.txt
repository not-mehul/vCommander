- Added Commission Tool
- Version check on startup with Github

- (New feature) Decommission tool CSV export shows the serial numbers alongside object ids wherever present.
- (New feature) Decommission tool now lists and deletes Access Station Pro devices
- (Bug fix) Alarms now creates a response, setting it to Self-monitored mode.
- (Bug fix) Decommission tool now deletes Alarm Input/Output devices.
- (Bug fix) AI powered search and certain additional analytics also toggled on by default
- (Bug fix) Alarms Specialist Commission will now allow toggling face analytics, with the default toggled to True.
- (Bug fix) Alarms Specialist creates a General Keycode for the Instructor - 123456
- (Bug fix) Session timeout is now a fixed 30-minute window from login.
